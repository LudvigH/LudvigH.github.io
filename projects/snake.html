Snake C++

  • Genre: 90’s Snake replica
  • Engine: C++ and sfml
  • Language: C++
  • Team size: 1
  • Duration: 1 weeks
  • Role: All

The idea was inspired by the game Cookie Clicker and the farming element in that game, and the story was heavily influenced by the movie the martian. Developed using Git and SourceTree with Scrum as methodology

In the end the game became a survival/crafting/collecting game, which of course was way to ambitious with the limited time we had, but at the same time a good learning experience.

Contributed with: main base(airlocks, rooms, satellite dish), Day/Night cycle, Environmental art, audio logs and associated systems.
